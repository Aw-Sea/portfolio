Блог для путешественников

О проекте

Полнофункциональная платформа для путешественников с возможностью создания маршрутов, добавления точек интереса и совместного планирования поездок.

Функционал:
  • Создание и редактирование маршрутов
  • Интерактивная карта с геолокацией
  • Система комментариев и оценок
  • Экспорт маршрутов в GPX формат
Технологии
Django 4.2 PostgreSQL Leaflet.js Bootstrap 5

Патриотический сайт

О проекте

Историко-патриотический портал для краеведческого проекта с акцентом на простоту управления контентом и современный дизайн.

Реализованный функционал:
  • Лента новостей и анонсов патриотических событий региона
  • Интерактивный каталог памятных мест с фотогалереями
  • Раздел с видеорепортажами и историческими хрониками
  • Адаптивный дизайн для всех устройств
Мои задачи:
  • Выбор и настройка CMS Grav для удобства управления
  • Кастомизация дизайна и интеграция медиа-функционала
  • Настройка интуитивно понятной админ-панели для заказчика

Grav CMS PHP Markdown HTML5 CSS3 JavaScript
