Traveler — сообщество путешественников

О проекте

Дипломный проект: полнофункциональная платформа для блогеров-путешественников с системой рекомендаций и социальными функциями.

Ключевые возможности:
  • ✅ Публикация постов с Rich-редактором (Froala Editor)
  • ✅ Лента подписок и персонализированные рекомендации
  • ✅ Социальные функции: лайки, комментарии, закладки
  • ✅ Система сообщений и уведомлений
  • ✅ Поиск и фильтрация по категориям
  • ✅ Смена темы (темная/светлая)
  • ✅ Адаптивный дизайн на Bootstrap
Технические особенности:
  • Система рекомендаций на основе активности пользователей
  • Модель UserActivity для анализа поведения
  • Админ-панель для модерации контента
  • Оптимизированные запросы к базе данных
Технологии
Django 4.2 PostgreSQL Bootstrap 5 Froala Editor JavaScript

Патриотический сайт

О проекте

Историко-патриотический портал для краеведческого проекта с акцентом на простоту управления контентом и современный дизайн.

Реализованный функционал:
  • Лента новостей и анонсов патриотических событий региона
  • Интерактивный каталог памятных мест с фотогалереями
  • Раздел с видеорепортажами и историческими хрониками
  • Адаптивный дизайн для всех устройств
Мои задачи:
  • Выбор и настройка CMS Grav для удобства управления
  • Кастомизация дизайна и интеграция медиа-функционала
  • Настройка интуитивно понятной админ-панели для заказчика
